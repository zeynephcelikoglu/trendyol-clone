# 🛍️ Trendyol Clone - React Native

Bu proje, Trendyol mobil uygulamasının temel işlevselliklerini taklit eden bir React Native uygulamasıdır. Kullanıcılar ürünleri listeleyebilir, detaylarını görebilir, sepete ekleyebilir ve uygulama içinde gezinme yapabilir.

## 🚀 Özellikler

- 🔐 **Giriş / Kayıt** sistemi
- 🏠 **Ana sayfa**: Öne çıkan ürünler, kampanyalar ve popüler markalar
- 🛍️ **Ürün listeleme** sayfası (Grid yapıda, responsive)
- 📄 **Ürün detay** sayfası (Fiyat, açıklama, puan vs.)
- 🛒 **Sepete ekleme & çıkarma** işlemleri
- 👤 **Profil ekranı** ve **çıkış yapma**
- 🔄 **React Navigation** ile tab navigasyon (Bottom bar)
- 🌐 **Fake Store API** ile gerçek ürün verileri

## 🧱 Kullanılan Teknolojiler

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/)
- [Context API](https://reactjs.org/docs/context.html) (Sepet yönetimi için)
- [FakeStoreAPI](https://fakestoreapi.com/) (Test verisi olarak)
